# quartz-crystal-frequency-meter
A low-cost microcontroller-based system to measure the resonant frequency of unknown quartz crystals using Arduino and MAX7219 display module

A cost-effective alternative to expensive laboratory instruments for 
measuring the resonant frequency of quartz crystals.

## How It Works
The system operates the microcontroller directly with the test crystal 
and compares its execution timing against a known internal reference 
to determine the crystal's frequency.

## Components Used
- Arduino (microcontroller)
- MAX7219 Display Module
- Quartz crystals (test samples)

## Features
- Low-cost hardware setup
- Minimal circuit complexity
- Real-time frequency display
- Built using open-source development tools
